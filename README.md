# FFmpeg-MX

Building Custom FFmpeg library for MX Player

## Usage

Download the codec file from the latest [Release](https://github.com/entr0pia/FFmpeg-MX/releases/latest/)

| Architecture | Code   |
| ------------ | ------ |
| arm64-v8a    | neon64 |
| armeabi-v7a  | neon   |
| x86          | x86    |
| x86_64       | x86_64 |
| all_in_one   | aio    |
